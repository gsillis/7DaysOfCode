# 7DaysOfCode
Repositório do projeto desenvolvido durante o desafio 7 Days Of Code da Alura. 

 - Arquitetura VIP-C
 - UI construídas programaticamente (view code)
 - Testes unitários
 - Request nativa com `async await` 
 - SPM 
 
### Bibliotecas terceiras:
- SnapKit
- KingFisher
